ESTRUCTURA BÁSICA DE HTML5

ESTRUCTURA ETIQUETAS ATRIBUTOS AUTOEVALUACION CREDITOS

ETIQUETAS

ATRIBUTOS DESCRIPCION
A
accept Lista de tipos aceptados por el servidor, usualmente un tipo de archivo
Accesskey Define una tecla que activa o agrega un foco al elemento.
action La URL del programa que procesa la información enviada en el formulario.
align Especifica el alineamiento horizontal del elemento.
alt Texto alternativo en caso de que la imagen no se pueda mostrar.
B
bgcolor Color de fondo del elemento
border El ancho del borde
C
charset/td> Declara la codificación de caracteres de la página o del script.
cite Contiene una URL que apunta a la fuente de la cita o cambio
class Usualmente empleado con CSS para aplicar estilos a elementos con propiedades en comun
cols Define el número de columnas en un área de texto
D
default Indica que la pista debe estar habilitada a menos que las preferencias del usuario indiquen algo diferente.
draggable Define si el elemento puede ser arrastrado.
F
for Describe elementos que pertenecen a éste.
H
height Ancho del contenedor e imagen
href La URL de un recurso asociado
I
icon Especifica una imagen relacionada con el comando.
id Identificador el elemento para acceder al mismo desde CSS o Javascript, este indicador debe ser unico para cada elemento.
L
list Identifica una serie de valores predefinidos para ser sugeridos al usuario.
lang Define el lenguaje utilizado en el elemento.
R
rows Define el número de filas en un área de texto.
rowspan Define el número de filas que una celda de una tabla debe abarcar.
S
selected Define un valor que será seleccionado al cargar la página.
shape Define una región seleccionable para hacer hipervínculos asociados con una forma asociada a una figura para crear un mapa.
size Define el ancho del elemento (en píxeles). Si el atributo del elemento es del tipo text o password entonces es el número de caracteres.
span Indica el número de columnas que se agrupan.
T
target Valores especiales: "_blank" indica una nueva ventana, "_parent" indica el marco padre del marco que contiene el código fuente, "_self" indica el marco donde está el código fuente y "_top" indica la ventana completa del navegador
