ESTRUCTURA BÁSICA DE HTML

ELEMENTOS SEMANTICOS:


HEADER:

Se utiliza para marcar la cabecera de una pagina, contiene el logotipo, una imagen y el cuadro de búsqueda. El componente mas importante es el TITTLE, esta etiqueta se muestra como identificador en la parte superior de la ventana del visualizador.

NAV

Representa una parte de una página que enlaza a otras paginas.

SECTION:

Representa un documento generico o de la seccion de aplicacion.

FOOTER:

Se utiliza para indicar el pie de pagina, el cual contiene informacion general acerca de su sección.

ASIDE

Representa una nota, un consejo, una explicación.

ARTICLE:

Representa una entrada independiente en un blog, revista,
